              # Road Accident Analysis (Excel)

## Overview
Road accident dataset analysis using Pivot Tables and Pivot Charts in Excel.

## Dataset
- Rows: 199
- Columns: 32

## Tools Used
- MS Excel (Pivot Table, Pivot Chart)

## Analysis
1. Accident Severity
2. Accidents by Day of Week
3. Cause of Accident

## Key Insights
- Slight Injury cases are highest.
- Monday has the highest accidents, Tuesday the lowest.
- Top causes include lane change, no distancing, and careless driving.
